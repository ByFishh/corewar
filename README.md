# Corewar
The goal of this project is to create a RedCode compiler and a Virtual machine.

## Installation
Clone the project.

```bash
cd corewar
make
./corewar/asm -h
./corewar/corewar -h
```

## Usage

```bash
./corewar/asm -h
./corewar/corewar -h
```
